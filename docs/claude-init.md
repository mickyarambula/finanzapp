# Finanzapp — Init prompt para Claude Code

Pega esto al inicio de cada sesión en Claude Code:

```
Lee estos archivos en orden y confirma que entendiste el contexto antes de hacer cualquier cosa:
1. docs/HANDOFF.md — qué se hizo en la última sesión y por dónde empezar
2. docs/ARCHITECTURE.md — estructura técnica del proyecto
3. docs/DECISIONS.md — decisiones tomadas y por qué
4. docs/ROADMAP.md — qué falta por hacer

Después de leerlos, dime en 3 líneas: dónde estamos, qué haremos hoy y si hay algo que deba saber antes de empezar.
```
